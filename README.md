# Student Performance Analysis Dashboard

## Project Overview
This project analyzes student academic performance to identify patterns and trends in examination scores.

## Tools Used
- Microsoft Excel
- PivotTables
- Charts

## Key Insights
- Female students performed better in reading and writing.
- Male students scored slightly higher in mathematics.
- Students who completed test preparation courses performed better overall.
- Parents level of education showed a positive relationship with students Performance.
